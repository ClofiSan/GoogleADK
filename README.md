# Arduino滑稽机器人

## 简介

采用ESP8266开发板作为主控，SG90舵机作为脖子，LED点阵作为笑脸组装出机器人，并利用Android APP实现控制，MQTT协议实现数据传输，可以做出简单的抬头，显示表情等简单动作。
